Pricing Web Component

A fully customizable, embeddable pricing component for your website

How to use this component

Simply include the script tag below in your HTML and use the custom element:

<script type="module" src="YOUR_NETLIFY_URL/pricing-component.js"></script>
<pricing-component currency="USD" highlight="pro"></pricing-component>

Additional Options

The component supports monthly/yearly billing toggle:

<pricing-component currency="USD" highlight="pro" billing-period="yearly" yearly-discount="20" ></pricing-component>

Created with Web Components. Free to use and customize.
